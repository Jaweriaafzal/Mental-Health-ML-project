# Mental Health ML Project

## 📌 Overview
This project is a Machine Learning-based Mental Health Prediction System.  
It analyzes mental health-related data and predicts mental health conditions using ML algorithms.

---

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

---

## 📂 Project Steps
1. Data Collection
2. Data Cleaning
3. Data Visualization
4. Model Training
5. Prediction and Evaluation

---

## 🤖 ML Algorithms Used
- Logistic Regression
- Decision Tree
- Random Forest

---

## 📊 Results
The model was trained and evaluated to predict mental health conditions with good accuracy.
